# Weather App

A simple desktop weather application built with Python and PyQt5.  
It fetches current weather data for any city using the OpenWeatherMap API and displays temperature, weather description, and an emoji.

## Features

- Enter a city name to get the current weather.
- Displays temperature in Fahrenheit.
- Shows a weather emoji and description.
- Handles common API and network errors gracefully.

## Requirements

- Python 3.x
- PyQt5
- requests

## Installation

1. Clone or download this repository.
2. Install dependencies:
   ```
   pip install PyQt5 requests
   ```

## Usage

1. Run the app:
   ```
   python weather app.py
   ```
2. Enter a city name and click "Get Weather".

## API Key

This app uses the OpenWeatherMap API.  
Replace the `api_key` variable in the code with your own API key if needed.
